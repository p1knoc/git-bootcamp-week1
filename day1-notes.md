# Day 1 Learning Notes

## Key COmmands Learned
- 'git init' - Initialized a repository
- 'git status' - Check repository state
- 'git add<file>' - State changes
- 'git commit -m "message"' - Save snapshot
- 'git log' - View history

## Important Concepts
- Working Directory: Where I edit files
- Staging Area: Where I prepare commits (like a loading dock)
- Repository: Where Git stores committed history

## Linux Integration
- Using nano editor in terminal
- Navigating with cd, mkdir
- Tab completion saves time
